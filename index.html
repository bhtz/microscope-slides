microscope.js

What is microscope.js ?

microscope.js is an open source nodejs MVC framework for productive web development. It is based on express.js and uses several popular npm modules like sequelize.js and passport.js.

All in the box !

microscope provided a number of tools for building modern web applications: jQuery, AngularJS, modernizr, stylus CSS pre-processor, toastr, bootstrap.

Command Line Tool

microscope provided a command line interface to quickly build web application. You can run server, generate some models or controllers and scaffold REST API, CRUD controller or Data Access Layer.
